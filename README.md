# 👋 Привет, я Аникин Сергей

Android-разработчик, увлечённый созданием удобных и производительных мобильных приложений. Специализируюсь на современной экосистеме Android с фокусом на архитектуру и кроссплатформенность.

### 🛠 Технологический стек

**Языки:**  
- **Kotlin** (основной) — корутины, Flow, потокобезопасность.

**Фреймворки:**
- **Jetpack Compose** — создаю адаптивные UI с использованием `ConstraintLayout`, пишу кастомные модификаторы, применяю `AnimatedVisibility` и анимации.
- **Kotlin Multiplatform** — выделяю бизнес-логику в shared-модуль, пишу платформозависимые обёртки для API и UI, обеспечиваю общую обработку ошибок.
- **Koin** — использую для внедрения зависимостей в Android-модуле; применяю `viewModelScope` и `get()` для получения экземпляров, настраиваю модули на уровне приложения и фич.
- **Room** — работаю с миграциями, использую аннотацию `@Relation` для связанных сущностей, интегрирую с `Flow` для реактивного доступа к данным.

**Архитектура:**  
- **Clean Architecture** — слои Domain/Data/Presentation.
- **MVVM** — State/Events, SavedStateHandle.
- **MVI** — для сложных экранов с реактивным состоянием.

**Инструменты:**  
- **Gradle** — кастомные задачи, оптимизация сборки.
- **Git** — rebase, cherry-pick, работа с конфликтами.
- **CI/CD** (GitHub Actions, GitLab CI) — пайплайны с unit-тестами, линтерами.
- **Firebase** — Crashlytics, Analytics, Remote Config.

- **UI-компоненты:** Активно использую Jetpack Compose для создания адаптивных и анимированных интерфейсов, пишу повторно используемые кастомные модификаторы.
